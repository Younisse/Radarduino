# Radarduino – Younisse AKBAB & Theo LASSAUNIERE - FR
Le Radarduino est un radar automatisé conçu pour détecter un objet en mouvement, mesurer sa vitesse et afficher cette vitesse en temps réel sur un écran intégré.
Grâce à un capteur de distance monté sur un servomoteur, le système effectue un balayage de l’espace, repère l’objet et calcule sa vitesse avec précision avant de l’afficher sur l’interface du radar.

# Radarduino – Younisse AKBAB & Theo LASSAUNIERE - EN
The Radarduino is an automated radar system designed to detect a moving object, measure its speed, and display that speed in real time on the integrated screen.
Using a distance sensor mounted on a servo motor, the system scans the area, detects the object, calculates its speed accurately, and then displays the result on the radar interface.
